Presentazione corso C#

Perché dovresti imparare C#?

Prima di tutto, perché C# è un potente e moderno linguaggio di programmazione object-oriented, uno dei più diffusi e richiesti al mondo. Con C# si possono sviluppare software indipendenti con una propria interfaccia utente cosi come possono essere sviluppate applicazioni web e applicazioni mobili. Si hanno a disposizione librerie e framework per lo sviluppo di qualsiasi software.

Quali tipi di applicazioni potrei sviluppare#?

Videogames (con Unity 3D) Mobile Apps per iOS/Android/Windows (con Xamarin) Web Apps (con ASP.NET, Blazor) Windows Apps (con il .NET Framework) Microservice, e altre applicazioni server-side su Mac/Linux/Win (con .NET Core)

Altre informazioni

Questo corso è una guida molto dettagliata del linguaggio C# 8, con l'utilizzo di .NET 3.1.
