Corsi - Programmazione in Python, C#, Html, SQL

Ciascun corso ha come obiettivo l'apprendimento delle conoscenze e lo sviluppo delle competenze di base del relativo linguaggio. Lo svolgimento di ciascun corso prevede un diverso numero di ore e, nel caso in cui almeno il 50% di esse siano state completate, il rilascio di un relativo attestato di frequenza.

I corsi disponibili riguardano quattro linguaggi, quali Python, C#, HTML e SQL.

  • Il corso di Python prevede 15 ore di lezione, durante le quali sarà utilizzata la piattaforma di Visual Studio Code. Per ulteriori dettagli, visitare la relativa pagina.
  • Il corso di C# prevede 18 ore di lezione, durante le quali sarà utilizzato l'ambiente di sviluppo Microsoft Visual Studio Community 2019. Per ulteriori dettagli, visitare la relativa pagina.
  • Il corso di Html prevede 12 ore di lezione, durante le quali sarà utilizzato Visual Studio Code. Per ulteriori dettagli, visitare la relativa pagina.
  • Il corso di SQL prevede 15 ore di lezione, durante le quali sarà utilizzato, in particolare, il DBMS MariaDB e la piattaforma Laragon. Per ulteriori dettagli, visitare la relativa pagina.
